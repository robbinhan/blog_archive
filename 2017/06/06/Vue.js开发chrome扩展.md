# Vue.js开发chrome扩展

好几年前做过一个小扩展，抓取文字然后在豆瓣搜索的。这次闲的无聊想做个hacker news的扩展，也整合了Vue体验下，感觉下来Vue确实比React更简单些。

## 安装vue-cli

```javascript
npm install -g vue-cli
```
## 生成项目

```javascript
vue init webpack my-project
```
## 安装依赖
```javascript
cd my-project
npm install
```
## 启动
```javascript
npm run dev
```
